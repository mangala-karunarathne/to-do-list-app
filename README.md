## Running Locally

1. Clone the repository

```bash
https://github.com/mangala-karunarathne/to-do-list-app.git
```

2. Install dependencies using pnpm

```bash
npm install
```

3. Copy the `.env.example` to `.env` and update the variables.

```bash
cp .env.example .env
```

4. Start the development server

```bash
npm start
```
